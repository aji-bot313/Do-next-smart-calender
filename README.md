# Smart Task Management System

A productivity-focused task management application designed to help users organize, prioritize, and complete their goals efficiently. The system combines intelligent automation with a user-friendly interface to create a seamless planning experience.

### Key Features

* **User Registration & Authentication** – Secure account creation and login system for personalized task management.
* **Dual-Mode Interface** – Switch between different viewing modes to suit user preferences and workflows.
* **Task Management** – Create, edit, delete, and organize tasks with ease.
* **Automatic Deadline Parsing** – Convert natural language deadlines (e.g., "next Friday at 5 PM") into structured due dates.
* **Priority Algorithm** – Automatically rank tasks based on urgency, importance, and approaching deadlines.
* **Persistent Storage** – Securely save tasks and user data for long-term access.
* **Visual Indicators** – Color-coded and status-based indicators to quickly identify task priority and completion progress.
* **Smart Reminders** – Receive notifications for upcoming deadlines and overdue tasks.
* **Terminal User Interface (TUI) Navigation** – Fast and keyboard-friendly navigation for power users.
* **Search & Filter System** – Quickly locate tasks using keywords, categories, priorities, or status filters.
* **Recurring Tasks** – Schedule daily, weekly, monthly, or custom recurring activities.
* **Streak Tracking & Motivation** – Build consistency by maintaining streaks for habits and daily goals, encouraging long-term productivity and discipline.

### Objective

The goal of this application is to provide a comprehensive productivity solution that not only helps users manage tasks effectively but also motivates them to develop positive habits through streak tracking, smart prioritization, and timely reminders.
